miniGallery

miniGallery is a simple gallery template for anyone out there wishing to use an image gallery which is non-reliant on javascript libraries and extremely light-weight.

The javascript used in miniGallery is the awesome Yetii script. Although Yetii is primarily a tab script, it's so flexible that is is easily customised to meet almost any requirements.

miniGallery template designed by: Jigowatt Premium Templates

The images used in this example are desktop backgrounds in Apple OSX.
